# Stethopy

* this project is ongoing

## Product:
A personal digital stethoscope that a patient can plug into their phone that transmits cardiovascular audio for telehealth professionals to diagnose with
Additional self-diagnostic capabilities within the app using machine learning to diagnose basic cardiac issues such as heart murmurs and arrhythmias

## App Features:
Profile page, Home Page, Listen Page, Telehealth/Messaging Page
